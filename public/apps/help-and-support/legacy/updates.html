Version 3.2.0 - The Immersion Update

March 13, 2026

This update delivers the most authentic Windows XP experience yet, with a complete cursor system overhaul, perfected Start Menu behavior, realistic app loading, and key fixes for native apps, ads, and stability.

    Native App & Performance Fixes
  • Fixed Installation/Updates Lag: Added a concurrency limit to prevent Android app/low end devices from throttling during VFS population.
  • Mandatory Update Overlay: Added a blocking Blue Screen-style overlay for outdated native apps (Electron and Android). Users are now clearly prompted to install the latest version.

    • Adware Popup System
  • Replaced intrusive full-screen video ads with authentic XP-style "Adware" dialog boxes.

    • Authentic Cursor System
  • The mouse cursor system has been completely overhauled to eliminate leaks from the host operating system. Your cursor will now remain a genuine Windows XP pointer (arrow, hand, working, or beam) at all times, providing a truly immersive experience. No more seeing your Mac or Windows cursor break the illusion!
  • Realistic "Working" Cursor: We've fine-tuned the "Working in Background" cursor (the arrow with the hourglass) to appear in all the right places, exactly like it did on the original. You'll now see it when loading applications (both for the first time and from memory), opening folders in Windows Explorer, navigating back and forward in your folder history, pasting, deleting, or creating new files, emptying the Recycle Bin or restoring files, applying a new visual theme or setting a new desktop wallpaper, refreshing the desktop, and logging off a user session.
  • Smart Application Loading: Apps now have a more realistic loading feel. The first time you open an application, it will have a slightly longer cold start delay, just like on a real PC. Subsequent launches of the same app will be much faster but still retain a subtle, snappy re-opening delay showing the working cursor to feel authentic.

    • Perfected Start Menu Behavior
  • The Start Menu has been re-engineered to behave identically to the original.
  • "All Programs" Locks Open: The "All Programs" list now correctly stays highlighted and open as you move your mouse into its submenus, only closing when you move back to your pinned apps.
  • Sticky Submenus: Folders inside "All Programs" (like "Accessories" or "Games") will now lock open and stay highlighted. They will only close when you deliberately move your mouse to a different folder at the same level, preventing accidental closures from stray mouse movements.
  • Stability: A rare bug where the Start button could get stuck in its active state without opening the Start Menu has been fixed.

    • Boot Screen Polish
  • Non-Draggable Boot Screen: The classic Windows XP boot animation is now locked to the screen, preventing it from being accidentally dragged around like a normal image.

Version 3.0.0 - The Unbound

February 25, 2026

This massive milestone represents a paradigm shift for Reborn XP, evolving from a web-based simulation into a multi-platform operating environment. Featuring a native Android application, a completely rewritten performance engine, and the long-awaited restoration of MSN Messenger, Version 3.0.0 is the definitive Reborn XP experience.

    Introducing Reborn XP for Android
  • Reborn XP is now available as a native Android app! Experience Windows XP on your phone with optimized touch controls, high-DPI scaling, and hardware-accelerated performance.
  • Unlike the web version, the Android app's Internet Explorer allows you to browse most of the websites without CORS or iframe restrictions, providing a truly open web experience.
  • A dynamic, multi-touch virtual gamepad overlay has been added for touch devices. Now you can play keyboard-based Flash games and App Market titles on Mobile.
  • Fully resolved coordinate synchronization and DPI scaling issues across various mobile resolutions and screen sizes.
  • Added dynamic touch controls for 3D Pinball to make the classic game fully playable on mobile devices.


    • MSN Messenger 7.5 Restoration
  • MSN Messenger 7.5 is back! We have restored the original UI, preloaded with classic Winks, Emoticons, Nudges, and authentic sounds.
  • Chat with pre-integrated NPC bots and explore preloaded conversation histories to relive the golden era of instant messaging.
  • For real-time user-to-user chat, Chatango app is now available via the App Market.

    • Desktop Logic & User Experience
  • Free Icon Positioning: Desktop icons have been converted to absolute positioning. You can now place your shortcuts anywhere on the screen.
  • Grid Snapping & Auto-Arrange: Implemented a professional 75x75px "Align to Grid" system. Enabling "Auto Arrange" or changing "Sort By" now intelligently calculates and saves a clean layout.
  • Every icon's coordinates and your specific view settings are now saved individually in localStorage for each user account.
  • Removed the banner advertisement from the logon screen to ensure a pure, nostalgic first impression.
  • Fixed the "Take a Tour" balloon popup; it now correctly tracks state per user and only appears during the very first logon of a new account.

    • Architectural & VFS Overhaul
  • Near-Instant Loading: Optimized the Virtual File System to make folder browsing and desktop initialization near-instant, regardless of file count.
  • Binary Streaming: Completely rewrote VFS logic to handle large Blobs as binary streams. This prevents main-thread freezing and eliminates errors during backup/restore.
  • Prevented the OS interface from freezing or flashing during application installations, file uploads, and ZIP extractions.
  • Added native-style dialogs with real-time progress bars for all file transfer operations between Reborn XP and the host OS.
  • Enabled in-session caching for UI assets (icons, sounds, system files). If your internet drops after boot, the OS remains functional and visually intact (no more broken images).

    • Application & System Updates
  • Implemented an auto-update system for installed apps. If a new version is released on the Market, you will be prompted to update directly without re-opening the App Market.
  • The "Qooqle" search engine has returned to Internet Explorer. It features a custom proxy for web, image, and video search, allowing you to play videos directly within search results (bypassing YouTube's iframe blocks).
  • Backups now use binary JSZip streaming. Automatically excludes the C:/RECYCLER folder to save space and includes a detection system to remain backward compatible with old backup formats.
  • The standalone Windows app has been upgraded to Electron 40 for improved security and performance.
  • Resolved titlebar style issues in the Royale Noir theme when windows are maximized.
  • Removed native Winamp (due to skins and file association issues)

    • App Market Changes
  • GTA: Vice City and Chatango are now available.
  • Removed Quake 3 (due to lack of touch controls); Winamp is planned to return as a 3rd party app soon.

Version 1.0 Series — The Foundation

April 22, 2021 - June 2024

    The Original Vision
  • The Reborn XP project began on April 22, 2021, as an open-source project to demonstrate the possibilities of vanilla JavaScript, HTML, and CSS. The initial goal was to create a visual simulation of the Windows XP desktop environment that could run directly in a web browser. For several years, it was hosted on GitHub and was most notably accessible via the rebornxp.js.org domain, prior to the acquisition of quenq.com in 2024. These early versions served as a crucial proof of concept, establishing the core desktop environment and the iconic Windows XP look.
  • The experience was centered around the classic Luna theme and a basic set of applications with limited interactivity. This foundational work was essential for building the community and vision that led to the complete transformation in Version 2.0.0.
