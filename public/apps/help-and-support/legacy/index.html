About Reborn XP

Reborn XP is an interactive simulation of the Windows XP operating system. It is a nostalgic passion project designed to recreate the look, feel, and functionality of one of computing's most iconic eras.

This project is not just a static app; it's a dynamic environment with a persistent virtual file system, a fully functional window manager, and a growing collection of applications that mimic their original counterparts.

Our goal is to provide a fun, educational, and entertaining experience. Whether you're here to relive memories, explore a classic OS for the first time, or see what's possible with modern web technologies, we're glad to have you.
